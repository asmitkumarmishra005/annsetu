

# 🍱 AnnSetu

**AnnSetu** is a web application that helps reduce food waste by connecting food donors with nearby NGOs. It provides a simple platform where users can donate surplus food, check food safety, locate NGOs, and track their donation.

## Features

* 🍱 Food Donation Form
* 🤖 AI Food Safety Assistant
* 📍 Nearby NGO Finder
* 🚚 Donation Tracking
* 📱 Responsive User Interface

## Technologies Used

* HTML
* CSS
* JavaScript
* GitHub Pages
